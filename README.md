
## Real time messaging system using laravel and vue


## Installation Instruction

	- Clone the repo.
	- Run 'composer install'
	- Run 'cp .env.example .env'
  	- Run 'php artisan key:genereate'
  	- Run 'php artisan serve'

	For Vue
		- Run 'npm install'
		- Run 'npm run dev'
	  	- Run 'npm run watch'


## WebSockets server
	- Run 'php artisan websockets:serve'

## Packages Used
	- laravel websocket
	- pusher-php-server
	- laravel echo
	- vue-chat-scroll
